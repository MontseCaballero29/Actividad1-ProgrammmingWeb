# Actividad 1 - Programación Web

Este repositorio contiene 10 ejercicios realizados para la asignatura de Programación Web.

En los ejercicios se utilizan las tecnologías básicas para el desarrollo web:

- HTML
- CSS
- JavaScript

El objetivo de esta actividad es practicar la estructura de páginas web, estilos visuales, formularios, tablas, enlaces, imágenes y el uso básico de JavaScript.
